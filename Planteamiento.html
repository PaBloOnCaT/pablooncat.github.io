2. PLANTEAMIENTO DEL PROYECTO

2.1. Justificación

La creación de una página web para IceCold surge como una necesidad para modernizar y optimizar la comercialización de productos en el sector de distribución de helados. Actualmente, muchas personas que administran negocios de venta de helados enfrentan dificultades para acceder a diversas marcas de manera eficiente.

Con esta plataforma, se facilitará la compra y distribución de helados, brindando una solución digital que ahorra tiempo y mejora la experiencia del cliente.

Desde el punto de vista organizacional, este proyecto permitirá a IceCold expandir su alcance, mejorar su posicionamiento en el mercado y ofrecer un servicio más ágil y accesible.

2.2. Objetivos

2.2.1. Objetivo general

Desarrollar una página web para la empresa IceCold con el fin de optimizar la comercialización y distribución de helados de diversas marcas, facilitando la compra para negocios del sector y permitiendo la personalización de pedidos según las preferencias de los clientes.

2.2.2. Objetivos específicos

  • Investigar: Las necesidades del mercado en la distribución de helados para definir los requerimientos de la plataforma.
  • Diseñar: La estructura y funcionalidades de la página web, asegurando una interfaz intuitiva y atractiva.
  • Desarrollar: La interfaz y la lógica del sistema utilizando tecnologías como HTML, CSS, Java, JavaScript y Oracle.
  • Implementar: El sistema IceCold web como base para la gestión y comercialización de productos.
  • Evaluar: Realizar pruebas unitarias y de integración para garantizar el correcto funcionamiento del sistema.

2.3. Alcance

El producto final de este proyecto será una plataforma web funcional que permitirá la comercialización de diversas marcas de helados, incluidas nuestra marca personal de helados personalizados.

La página ofrecerá una interfaz amigable y eficiente para que negocios y comerciantes puedan adquirir productos sin necesidad de desplazarse físicamente.

2.4. Metodología

  • Investigación y planificación: Análisis del mercado para identificar necesidades de clientes y definir requisitos.
  • Diseño de la plataforma: Creación de prototipos con estructura e interfaz intuitiva.
  • Desarrollo e implementación: Programación e integración del sistema con bases de datos y opciones de pago.
  • Pruebas y ajustes: Evaluación de rendimiento, seguridad y usabilidad.
  • Lanzamiento y seguimiento: Monitoreo continuo para optimizar la experiencia de compra.
